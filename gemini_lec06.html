生成AI（Gemini）講座

〜生活を豊かに楽しくする24の魔法〜

AI（人工知能）である「Gemini」を、皆様の頼もしい相棒として日常生活で活用する方法を学びます。難しく考える必要はありません。普段、人に話しかけるような言葉で、AIとの会話を楽しんでみましょう。

【第1回】AIを健康と毎日の食事に活かそう

本日の目標： AIを使って、献立の悩みや日々の健康管理をサポートしてもらう感覚を掴む。

1. 冷蔵庫の余り物で献立作り

  • 基本演習:
    「冷蔵庫にナスと鶏むね肉、卵があります。これらを使って、高齢者でも食べやすい柔らかいおかずのレシピを1つ教えてください。作り方の手順も短く簡単に説明して。」
  • 発展課題: 「この料理の完成イメージ画像を生成して」と頼み、レシピカードを作ってみましょう。
  • 便利な保存方法: スマホのメモ帳アプリに「AIレシピ集」フォルダを作り、テキストと画面のスクリーンショット（画面保存）を保存します。

2. 薬の飲み合わせや副作用の簡易確認

  • 基本演習:
    「（薬の名前）という薬を処方されました。一緒に食べてはいけないものや、気をつけるべき体調の変化を、箇条書きでわかりやすく教えて。」（※最終確認は医師へ）
  • 発展課題: 冷蔵庫に貼れるよう、大きな文字と短い注意書きのポスター形式で出力させてみましょう。
  • 便利な保存方法: 印刷して「お薬手帳」に挟むか、LINEの「Keep」機能で保存します。

3. 自宅でできる軽い運動プログラム

  • 基本演習:
    「椅子に座ったままできて、膝に負担をかけずに足の筋肉を鍛えられる5分間の体操メニューを考えてください。」
  • 発展課題: 「1ヶ月間、毎日違うメニューをこなすための30日分の表を作って」と頼み、日めくりカレンダーにしてみましょう。
  • 便利な保存方法: 表を印刷して壁や冷蔵庫に貼り、実践した日に丸をつけて記録します。

4. 栄養バランスのチェック

  • 基本演習:
    「今日のご飯は朝：食パン、昼：そば、夜：焼き魚とご飯と味噌汁でした。シニアとして不足している栄養素と、それを補うおすすめ食材を教えて。」
  • 発展課題: 「不足を補うための、明日のスーパーでの買い物メモ（チェックリスト）を作って」とお願いしてみましょう。
  • 便利な保存方法: 買い物リストの画面をスクリーンショットし、スーパーですぐ見られるようにします。

💡 第1回のまとめ

  • AIは毎日の献立作りや運動メニューの考案など、健康管理の心強い味方になります。
  • 薬の情報などは参考に留め、最終的な判断は必ず医師や薬剤師に確認しましょう。
  • 得られた結果をメモやカレンダーに残す工夫で、生活習慣として定着しやすくなります。

【第2回】AIで日常の疑問を解決し、学びを深めよう

本日の目標： 身近なITの疑問や、ニュース、歴史など、知的好奇心を満たす使い方を学ぶ。

1. スマホや家電の「これどうやるの？」解決

  • 基本演習:
    「スマホのLINEで友達から送られてきた写真を保存して後で見られるようにしたいです。操作手順を優しく教えてください。」
  • 発展課題: 「ステップごとに図解を入れるための、スライド資料の構成案にして」と頼み、マニュアル化の土台を作ります。
  • 便利な保存方法: 回答を印刷し、機器のそばに「我が家のITマニュアル」として常備します。

2. 歴史や文化の深掘り

  • 基本演習:
    「戦後の東京オリンピック（1964年）の頃、日本の街並みや人々の暮らしはどんな様子でしたか？当時の流行なども含めて教えてください。」
  • 発展課題: 得られた回答を「小学生の孫に話して聞かせるための紙芝居の台本にして」と変換してみましょう。
  • 便利な保存方法: パソコンのWordなどに貼り付けて「自分史」やエッセイのネタとして保存します。

3. 最新ニュースの要約

  • 基本演習:
    「今話題の『（ニュースのキーワード）』について、何が起きているのかを小学生にもわかるような言葉で3つのポイントにまとめて。」
  • 発展課題: 「『週刊・〇〇（自分の名前）新聞』というタイトルで、新聞記事風のレイアウトで出力して」と頼んでみましょう。
  • 便利な保存方法: スマホのメモ帳に保存し、ご友人との集まりの前の話題作りとして見返します。

4. 若者言葉や流行の解説

  • 基本演習:
    「最近よく聞く『タイパが良い』という言葉はどういう意味ですか？なぜ若い人たちの間で流行っているのか教えてください。」
  • 発展課題: 学んだ言葉を使った「LINEスタンプ用の画像」をAIに描かせてみましょう。
  • 便利な保存方法: 手書きの小さなノートに「現代用語辞典」として書き写すのも良い脳トレになります。

💡 第2回のまとめ

  • スマホの操作や気になるニュースなど、わからないことはすぐにAIに聞いて解決しましょう。
  • 難しい言葉は「小学生にもわかるように」など、自分に合ったレベルで解説してもらえます。
  • 昔の出来事などを深掘りして、家族や友人との会話の楽しい「ネタ帳」として活用しましょう。

【第3回】AIと一緒に趣味と旅行をもっと楽しもう

本日の目標： 旅行計画や読書、園芸など、自分の趣味をAIの力でさらに充実させる。

1. ゆったり楽しむ旅行プランの作成

  • 基本演習:
    「夫婦二人で京都へ1泊2日。あまり歩かずタクシー移動中心で、景色と食事が楽しめるシニア向け観光プランを作って。」
  • 発展課題: プランに合わせて「シニア夫婦が持っていくべき持ち物チェックリスト」も作成してもらいましょう。
  • 便利な保存方法: 作成したプランやリストは、そのままLINEで同行者に送信（共有）して残します。

2. ガーデニングのアドバイス

  • 基本演習:
    「庭のアジサイの剪定は、いつ、どのあたりを切れば良いですか？初心者にもわかるようにコツを教えて。」
  • 発展課題: 「アジサイの1年間の水やり、肥料、剪定のタイミングを月別の表にして」と頼み、年間カレンダーを作ります。
  • 便利な保存方法: カレンダーを印刷し、園芸用品の箱や物置の壁に貼り付けておきます。

3. 読書や映画のレコメンド

  • 基本演習:
    「司馬遼太郎の歴史小説が好きです。最近の本や映画の中で、私の好みに合いそうなおすすめを3つ教えてください。」
  • 発展課題: おすすめされた作品を「図書館で予約しやすいようにタイトルと作者名だけのリストにして」と頼みましょう。
  • 便利な保存方法: スマホのメモ帳に「読みたい本・映画リスト」を作り、随時追加していきます。

4. 俳句や短歌の添削・相談

  • 基本演習:
    「俳句を作りました。『（自分の句）』。良いところと、もっと季節感を出すための言葉のアドバイスをください。」
  • 発展課題: 句の情景に合わせた水墨画風の画像をAIに生成させ、テキストと組み合わせてみましょう。
  • 便利な保存方法: 画像編集アプリを使って「写真＋俳句」の画像作品にし、スマホの壁紙に設定します。

💡 第3回のまとめ

  • 旅行の計画からガーデニングのコツまで、自分の趣味をより深く楽しむためにAIを活用できます。
  • 自分の好みをAIに伝えておすすめの作品を教えてもらい、新しい趣味の世界を広げましょう。
  • 画像生成機能などを組み合わせることで、俳句などの趣味がさらに素敵な「作品」に仕上がります。

【第4回】AIで創造力を発揮しよう！作品作りに挑戦

本日の目標： AIの文章作成能力や画像生成能力を使って、世界に一つだけの「作品」を作る。

1. 孫が喜ぶ「読み聞かせ」の物語作り

  • 基本演習:
    「5歳の孫の『〇〇くん』が主人公で、しゃべるウサギと魔法の木の実を探す、3分で読み聞かせができるお話を作って。」
  • 発展課題: 各シーンの挿絵をAIに描かせ、タブレットで読める「デジタル絵本」に挑戦しましょう。
  • 便利な保存方法: 文章は印刷し、挿絵と組み合わせたものはPDF形式などで保存します。

2. 自分史の作成サポート

  • 基本演習:
    「子供時代の思い出を文章にしたいです。昭和30年代の田舎の暮らしについて質問してください。答える形で自分史を書きます。」
  • 発展課題: まとまった文章に「目次と章タイトル」を付けて、本のような構成に整えてもらいましょう。
  • 便利な保存方法: パソコンのWordなどで保存し、USBメモリにもバックアップを取った上で印刷・製本します。

3. 脳トレ・クイズの出題と作成

  • 基本演習:
    「脳トレのために、少し難しい漢字の読み方クイズを5問出してください。1問ずつ出して解説も教えて。」
  • 発展課題: 「このクイズを20問まとめて、印刷して書き込めるドリル形式（回答は別ページ）で出力して」と頼んでみましょう。
  • 便利な保存方法: ドリル形式のテキストを印刷し、机の上に置いて毎日の日課にします。

4. AIで画像生成（ポストカード作り）

  • 基本演習:
    「日本の春を感じる水彩画のような絵を描いて。内容は『満開の桜、静かな川、遠くに富士山』で。」
  • 発展課題: スマホの「待ち受け画面のサイズ」に合わせて画像を再生成させてみましょう。
  • 便利な保存方法: 画像をスマホやパソコンの「写真・ピクチャ」フォルダに保存し、オンライン印刷サービスでグッズ化するのもおすすめです。

💡 第4回のまとめ

  • 孫への絵本や自分史の作成など、AIを優秀なアシスタントにして世界に一つだけの作品を作れます。
  • 画像生成機能を使えば、頭の中のイメージを言葉で伝えるだけで本格的な絵やポストカードを作成できます。
  • 作成したクイズドリルなどは、自分だけでなく家族と一緒に楽しむコミュニケーションツールにもなります。

【第5回】AIで広がるコミュニケーションと社会との繋がり

本日の目標： 人間関係を円滑にし、社会と繋がるためのツールとしてAIを活用する。

1. 丁寧な手紙やメールの代筆

  • 基本演習:
    「お世話になった先生に、近況報告と季節の挨拶状を送りたいです。丁寧で温かみのある春の手紙の例文を書いて。」
  • 発展課題: 春の挨拶状の背景に使える「淡い桜色の便箋の画像」をAIに作らせ、デジタルグリーティングカードにしてみましょう。
  • 便利な保存方法: よく使う挨拶文は、スマホの「ユーザー辞書」に登録しておくか、メールの下書きに保存します。

2. 地域のボランティアやサークル活動のアイデア

  • 基本演習:
    「編み物が得意です。この特技を活かして地域の人と交流できるボランティアやサークル活動にはどんなものがありますか？」
  • 発展課題: 「公民館に貼るための、編み物サークルメンバー募集ポスターのレイアウト案を出して」と依頼してみましょう。
  • 便利な保存方法: AIの提案をWordなどに貼り付け、実際の活動準備やチラシ作りの土台として活用します。

3. プレゼント選びの相談

  • 基本演習:
    「中学生の孫娘に、人気で実用的な予算1万円以内の入学祝いをいくつか提案して。」
  • 発展課題: 提案された候補を「メリット・価格・おすすめ理由」の表形式にまとめ、家族会議の資料にしてみましょう。
  • 便利な保存方法: 比較表をスクリーンショットして家族のLINEグループに共有します。

4. 話し相手・相談相手として

  • 基本演習:
    「今日、綺麗な青い鳥を見かけて幸せな気分になりました。何かこの出来事に関連する楽しい話題を振ってくれませんか？」
  • 発展課題: これまでのAIとの会話から「私が元気になるようなポジティブな言葉を31個選んで、日めくりカレンダーにして」と頼んでみましょう。
  • 便利な保存方法: 手書きの日記帳に、面白かったAIの回答や気づきを一行だけ書き留めて残しておきます。

💡 第5回のまとめ

  • 手紙の代筆やプレゼント選びなど、周囲の人との関わりをより豊かで丁寧なものにするためにAIを利用できます。
  • 自分の特技を活かしたボランティア活動のアイデアをもらい、地域社会との繋がりを広げるきっかけにしましょう。
  • 日常のちょっとした出来事や感情を話す相手として、AIとの気軽な会話を楽しんでみてください。

【第6回】AIを自分だけの専属アシスタントに育てる（発展・総集編）

本日の目標： 学んだテクニックを組み合わせ、AIを「自分だけの専属の相棒」としてさらに高度に使いこなす。

1. 複数の条件を組み合わせた「複合プロンプト」

これまでの要素を組み合わせ、一回の質問で多くの情報を引き出します。

  • 基本演習:
    「来月、青森へ旅行に行きます。『おすすめの観光ルート』『名物であるりんごの歴史』『疲れたときにホテルでできる簡単なストレッチ』の3つを同時に教えてください。」
  • 発展課題: 「この内容を、旅行に持っていくための1枚のA4サイズのチラシ風にレイアウトして文章を整えて」と頼んでみましょう。
  • 便利な保存方法: 回答をWordに貼り付けて、オリジナルの旅行ガイドブックとして印刷します。

2. AIに役割を与えて相談する（ロールプレイ）

AIに「専門家」になりきってもらうことで、より質の高いアドバイスをもらいます。

  • 基本演習:
    「あなたは『シニア向けのプロの健康アドバイザー』です。その立場から、毎日白米をたくさん食べてしまう私に、無理なく糖質を減らすアドバイスを優しくしてください。」
  • 発展課題: 「今度は『辛口の料理評論家』になりきって、私の作った肉じゃが（少し味が薄かった）の改善点をユーモアたっぷりに指摘して」と楽しんでみましょう。
  • 便利な保存方法: 面白かった会話のやり取りをスクリーンショットで残し、ご家族にシェアして笑いの種にします。

3. 前の会話の続きをする（コンテキストの活用）

AIは以前の会話の流れを覚えています。何度も会話のキャッチボールをしてみましょう。

  • 基本演習:
    （第1回で作った献立の会話の続きで）「昨日教えてもらった鶏肉とナスの料理、とても美味しかったです！今日は豚肉があるのですが、昨日の料理と味が被らない和風のおかずを教えて。」
  • 発展課題: 「この1週間の献立のやり取りを踏まえて、私の好みの味付けの傾向を分析してみて」と頼み、AIに自分を理解させましょう。
  • 便利な保存方法: 同じチャット（会話の部屋）を消さずに使い続けることで、AIがあなたの好みをどんどん学習していく自分専用のデータベースになります。

4. 【総集編】私だけのAI活用プラン作成

これまでの集大成として、自分専用のAI活用計画をAI自身に作ってもらいます。

  • 基本演習:
    「私は70代で、趣味は写真と散歩、パソコンは少し使えます。全6回のAI講座を修了した私が、今後1年間でAIを使って毎日を楽しくするための『私だけのAI活用プラン・10カ条』を作ってください。」
  • 発展課題: 「その10カ条を達成するために、私が今週まずやるべき『小さな第一歩』を3つ提案して」と背中を押してもらいましょう。
  • 便利な保存方法: 完成した「10カ条」を色紙などに筆ペンで書き写し、パソコンの横や目につく場所に飾って、AIとの新しい生活の宣言とします。

💡 第6回のまとめ

  • 複数の条件を組み合わせたり、AIに専門家の役割を与えたりすることで、より高度で具体的な回答を引き出せます。
  • 会話を継続することで、AIはあなたの好みや状況を学習し、ますます自分に合った「パーソナルな相棒」へと育ちます。
  • 最後に作成した「私だけのAI活用プラン」を道しるべに、これからも自由な発想でAIとの生活を楽しんでいきましょう。
